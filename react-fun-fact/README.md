# React Fun Facts
A static React application that introduces the core fundamentals of React, including JSX, component structure, and styling. This project focuses on building reusable components and organizing a simple UI, serving as a foundation for understanding how React renders and structures applications.

## What I learned
- How to set up and structure a basic React application
- Writing and using JSX to describe UI
- Creating and organising functional components
- Applying styles in React (CSS + inline styling)
- Understanding how React renders components to the DOM
